Requerimientos del Proyecto

Tipo Descripción
Funcional El sistema debe permitir a los usuarios registrarse y autenticarse.
Funcional El sistema debe permitir a los usuarios privatizar sus resultados si no quieren ser listadas.
Funcional El sistema debe permitir a los ver las puntuaciones de los demas siempre que sea posible.
No Funcional El sistema debe ser accesible desde dispositivos móviles.
No Funcional El sistema debe responder en menos de 2 segundos para cualquier solicitud.
No Funcional El sistema debe ser seguro y proteger los datos de los usuarios (correos de registro).
